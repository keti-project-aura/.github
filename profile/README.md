# On-Device AI Cooperative Actions Framework
온디바이스 AI 기반 자율협업 IoT 핵심 기술 개발 (세부2) 공개 소프트웨어 레포지토리

본 저장소는 과학기술정보통신부 정보통신·방송 기술개발사업  
“온디바이스AI 기반 자율협업 IoT 핵심 기술개발” (과제번호 RS-2025-02263031)의  
공개 소프트웨어 성과를 관리·운영하기 위해 구축되었습니다.

## 과제 목표

본 프로젝트는 다음 목표를 지향합니다:

- 온디바이스 AI 기반 이기종 사물 간 협력 인지·판단·행동 기술 개발
- 실환경에서 실행 가능한 자율 협업 IoT 프레임워크 구축
- 로봇·드론·센서·카메라 등 다양한 사물 기반 서비스 실현
- 오픈소스 공개를 통한 산업·학계 생태계 활성화
